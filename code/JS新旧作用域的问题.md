```
var a = 10
{
    a = 99
    function a(){}
    a = 30
}
console.log(a); // ？
```

编译后的真正执行
```
var a¹ = 10;
 {
   function a²() {} // 函数提升
   a² = 99;
   a¹ = a²; // function a²() {}函数的声明语句执行，将内部的a值同步到外部的a（变量突破）此时外部的a和内部的a值都为99。  
   a² = 30;
}
console.log(a¹);
```
在块级作用域中声明的函数，会函数提升到作用域顶部，并且执行到函数的声明语句时，会将其值同步到外层作用域。
